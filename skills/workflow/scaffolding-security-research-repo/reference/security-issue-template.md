> **STOP.** This project asks that vulnerabilities NOT be reported in public
> GitHub issues.
>
> If you believe you have found a security vulnerability:
>
> 1. Close this issue without filling it in.
> 2. Email **<security_contact>** with a description, repro steps, and the
>    version you tested against.
> 3. See [SECURITY.md](../../SECURITY.md) for our response SLA and
>    [VDP.md](../../VDP.md) for safe-harbor language and the coordinated-
>    disclosure window.
>
> If you are NOT reporting a security vulnerability and got here by mistake,
> please open a regular issue using the "Bug report" or "Feature request"
> template instead.
